# Moduł JavaScript
## Lekcja 4
### Zadanie 1 - poziom podstawowy 
Stwórz funkcję `fillElement`, która otrzyma w argumencie dowolny `string`. Funkcja ma za zadanie utworzyć element `<span>` z pomocą `document.createElement`, umieścić w tym spanie tekst przekazany w parametrze funkcji, oraz ma zwrócić ten element.

Przykładowy input:
`fillElement('Przykładowy tekst');`

Przykładowy output:
element `<span>Przykładowy tekst</span>`
